# 1.3.1 What's Hosting?

**Web Hosting** is the service that makes a website or application accessible on the Internet.

* **Definition:** It involves storing the files, code, and databases of a website on a powerful computer called a **web server**.
* **Service Provider:** A **Web Host** is a company that owns and maintains these servers and provides the technology and connectivity needed for the website to be viewed by anyone, anywhere.
* **Backend Relevance:** Backend code (like Node.js/Express) is deployed to a hosting environment (e.g., a cloud server like AWS, DigitalOcean, or a dedicated VPS) to run and be accessible over the public Internet.
